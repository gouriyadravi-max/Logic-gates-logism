# Logic Gates using Logisim

## Objective

The objective of this project is to design and simulate basic digital logic circuits using Logisim. The project aims to understand the working of fundamental logic gates and combinational circuits by implementing and testing them in a digital simulation environment.

## Circuits Implemented

- AND Gate
- OR Gate
- NOT Gate
- NAND Gate
- NOR Gate
- XOR Gate
- XNOR Gate
- Half Adder

## Software Used

- Logisim

## Outcome

Successfully designed and verified the operation of all basic logic gates and a Half Adder using Logisim, improving the understanding of digital logic design.
